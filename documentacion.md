# 📘 Documentación SmartBudget

## 1. Descripción

SmartBudget es una interfaz web para visualizar finanzas personales de forma moderna y sencilla.

---

## 2. Metodología CSS

Se utilizó la metodología BEM para mantener clases organizadas, escalables y reutilizables.

---

## 3. SASS (Arquitectura 7-1)

Se implementó SASS con estructura modular:

- abstracts
- base
- components
- layout
- pages
- themes
- vendors

Esto permite escalabilidad y orden del proyecto.

---

## 4. Bootstrap 4

Se utilizó Bootstrap 4 para acelerar el desarrollo del layout y asegurar responsividad.

---

## 5. Responsive Design

El sitio es completamente adaptable a:

- móviles
- tablets
- desktop

Usando media queries y sistema de grid.

---

## 6. Conclusión

El proyecto cumple con los requerimientos de una interfaz moderna, escalable y profesional.
