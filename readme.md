# 💰 SmartBudget

SmartBudget es una landing page moderna de una aplicación de finanzas personales que permite visualizar ingresos, gastos y presupuesto de forma simple, rápida y visual.

---

## 🚀 Demo

https://tu-usuario.github.io/SmartBudget

---

## 🧠 Tecnologías utilizadas

- HTML5 semántico
- CSS3 moderno
- SASS (arquitectura 7-1)
- Bootstrap 4
- Flexbox & Grid
- Diseño responsive
