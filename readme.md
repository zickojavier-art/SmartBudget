# 💰 SmartBudget

SmartBudget es una landing page de una aplicación de finanzas personales.

## 🚀 Descripción

Este proyecto permite visualizar ingresos, gastos y presupuestos de forma simple y moderna.

## 🧠 Tecnologías utilizadas

- HTML5
- CSS3
- SASS
- Bootstrap 4

## 🎯 Funcionalidades

- Diseño moderno tipo SaaS
- Interfaz responsive
- Secciones organizadas
- Uso de componentes Bootstrap

## 📱 Responsive

✔ Mobile  
✔ Tablet  
✔ Desktop

## 👨‍💻 Autor

Proyecto realizado como evaluación de desarrollo frontend.
